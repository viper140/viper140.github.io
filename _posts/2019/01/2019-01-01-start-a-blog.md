jekyll、hexo、hugo 怎么选？

<!--more-->

## 1 对比

- jekyll 使用 ruby 编写

  - 优点：github 官方集成，本地无需任何环境，push 的时候，构建的活儿 github 包办
  - 缺点：github 官方提供的主题挺丑

- hexo 使用 nodejs 编写

  - 优点：nodejs 对前端友好，主题较多，构建速度介于 hugo 和 jekyll 之间
  - 缺点：麻烦；前端朋友说厌倦了 node_module

- hugo 使用 go 编写

  - 优点：构建速度快
  - 缺点：麻烦
